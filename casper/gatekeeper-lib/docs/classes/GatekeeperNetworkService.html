Class GatekeeperNetworkService

Encapsulates the actions performed by a gatekeeper network authority. This is actually not required we think given the "gatekeeper" functionality is handled by the whitelisting feature in the KYC Token contract. The "master" key is the only one allowed to update the minters array. This "master" key needs to be owned by either Civic or Casper!
